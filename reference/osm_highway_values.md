# Values for the 'highway' tag in OpenStreetMap data

`osm_highway_values` contains a character vector of values for the
'highway' tag in OpenStreetMap data.

## Usage

``` r
osm_highway_values
```

## Format

A character vector of length 12.

## Examples

``` r
osm_highway_values
#>  [1] "motorway"       "trunk"          "primary"        "secondary"     
#>  [5] "tertiary"       "unclassified"   "residential"    "motorway_link" 
#>  [9] "trunk_link"     "primary_link"   "secondary_link" "tertiary_link" 
```
